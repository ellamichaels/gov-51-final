Legal Context - Utah

Non-Custodial Promise to Appear Instead of Custodial Arrest

  * Citation in Lieu of Arrest

    * Can citations be issued as an alternative to custodial arrest for criminal
      offenses?

      - Yes. [Utah Code § 77-7-18 (2012)]

    * Offenses eligible for citations instead of arrest:

      - Infractions. [Utah Code § 77-7-18 (2012)]
      - All misdemeanors. [Utah Code § 77-7-18 (2012)]

    * Circumstances under which citations shall not be issued:

      - None. [Utah Code § 77-7-18 (2012)]

    * Are citations presumptive for eligible offenses?

      - No. [Utah Code § 77-7-18 (2012)]

    * Authorities who may issue citations:

      - Any law enforcement officer. [Utah Code § 77-7-18 (2012)]
      - Other authorized personnel. [Utah Code § 77-7-18 (2012)]

  * Sentencing Options

    * Legal status of marijuana:

      - Possession of marijuana is illegal. [Utah Code § 58-37-8(1)(a) (2012)]

  * Clearing a Criminal Record

    * Circumstances under which an arrest record may be cleared (qualification
      requirements may be satisfied by different combinations of the criteria):

      - Please note, the law(s) governing expungement have changed since the
        time of this study period. Please refer to the current statutes for most
        up-to-date information.. [Utah Code Ann. § 77-40-104]
      - The petitioner obtains a certificate of eligibility. [Utah Code Ann. §
        77-40-104 (2012)]
      - At least 30 days have passed since the arrest. [Utah Code Ann. §
        77-40-104 (2012)]
      - The petitioner is not involved in any pending criminal procedures. [Utah
        Code Ann. § 77-40-104 (2012)]
      - The prosecutor has made a final determination that no charges will be
        filed. [Utah Code Ann. § 77-40-104 (2012)]
      - The case was dismissed with prejudice. [Utah Code Ann. § 77-40-104
        (2012)]
      - The petitioner was acquitted at trial on all charges where expunction is
        sought. [Utah Code Ann. § 77-40-104 (2012)]
      - The statute of limitations has expired on all charges where expunction
        is sought. [Utah Code Ann. § 77-40-104 (2012)]

  * Age & Charging

    * Minimum age at which an individual can be prosecuted as an adult:

      - 16. [Utah Code § 78A-6-701(1) (2012)]

    * Age at which an individual can no longer be considered a juvenile:

      - 18. [Utah Code § 78A-6-105(4) (2012)]

    * Circumstances under which a conviction record may be cleared
      (qualification requirements may be satisfied by different combinations of
      the criteria):

      - Please note, the law(s) governing expungement have changed since the
        time of this study period. Please refer to the current statutes for the
        most up-to-date information.. [Utah Code Ann. § 77-40-105]
      - A petition or request has been filed and accepted. [Utah Code Ann. §
        77-40-105 (2012)]
      - The petitioner obtains a certificate of eligibility. [Utah Code Ann. §
        77-40-105 (2012)]
      - The petitioner is not involved in any pending criminal procedures. [Utah
        Code Ann. § 77-40-105 (2012)]
      - The petitioner has no outstanding fines, interest, or restitution
        orders. [Utah Code Ann. § 77-40-105 (2012)]
      - The conviction for which expungement is sought is not for a capital
        felony, 1st degree felony, violent felony, felony automobile homicide,
        felony DUI offense, or a registerable sex offense. [Utah Code Ann. §
        77-40-105 (2012)]
      - A period of time has elapsed from the date the petitioner was convicted
        or released from incarceration, parole or probation. The length of this
        time period depends on the severity of the offense for which expunction
        is sought.. [Utah Code Ann. § 77-40-105 (2012)]
      - The institution authorizing expunction has determined that the
        petitioner's criminal record, including previously expunged convictions,
        meets eligibility requirements (see statutes for details). [Utah Code
        Ann. § 77-40-105 (2012)]

    * Circumstances under which pardons may be granted:

      - An application or petition has been submitted and accepted. [Utah Admin.
        Code r. R671.315 (2013),Utah Code Ann. § 77-27-5 (2012)]
      - Approval from the governor. [Utah Admin. Code r. R671.315 (2013),Utah
        Code Ann. § 77-27-5 (2012)]
      - Majority approval from the board of pardons. [Utah Admin. Code r.
        R671.315 (2013),Utah Code Ann. § 77-27-5 (2012)]
      - Five years have passed since the sentence for the conviction has been
        terminated or expired. [Utah Admin. Code r. R671.315 (2013),Utah Code
        Ann. § 77-27-5 (2012)]
      - A pardons hearing must be held in open session following due public
        notice. [Utah Admin. Code r. R671.315 (2013),Utah Code Ann. § 77-27-5
        (2012)]
      - For temporary reprieves of convictions by the Governor, the offense must
        not be for treason or involve impeachment. [Utah Admin. Code r. R671.315
        (2013),Utah Code Ann. § 77-27-5 (2012)]

    * Do expunged records appear in public record requests?

      - No. [Utah Code § 77-40-110 (2012)]

      - Deportation. [Utah Code § 76-9-1006 (2012)]

Nonviolent Misdemeanor Cases with Nonmonetary Release (ROR)

  * Pretrial Release Eligibility

    * Offenses presumptively denied or ineligible for pretrial release:

      - Capital offenses. [Utah Code § 77-20-1(2) (2012)]
      - Felonies committed while on probation or parole. [Utah Code § 77-20-1(2)
        (2012)]
      - Felonies when there is substantial evidence to support the charge and
        the court finds by clear and convincing evidence that the person would
        constitute a substantial danger to any other person or to the community,
        or is likely to flee the jurisdiction of the court, if released on bail.
        [Utah Code § 77-20-1(2) (2012)]

    * Does state law require the use of a risk assessment tool in deciding
      pretrial release?

      - No. [Utah Code § 77-20-1(5) (2012)]

  * Pretrial Release Conditions

    * Does state law ban the use of cash bail as a condition for pretrial
      release for certain offenses?


    * Non-financial conditions of pretrial release authorized by state law:

      - Promise to appear in court (release on own recognizance). [Utah Code §
        77-20-3 (2012)]
      - Any conditions necessary to ensure the defendant's appearance in court
        and public safety. [Utah Code § 77-20-1(3) (2012)]

  * Pretrial Detention

    * Are law enforcement officials permitted to make a pretrial release
      decision when a bail hearing cannot be promptly held?

      - No. [Utah R. Crim. P. 7(c)(1)]

    * Offenses that require a hearing to deny pretrial release:

Pretrial Diversion of Nonviolent Misdemeanors

  * Pretrial Diversion

    * Individuals responsible for pretrial diversion admission decisions (may be
      program-specific):

      - Judges. [Utah Code § 77-2-5(1), (4) (2012),Utah Code § 77-2a-3(6)
        (2012)]
      - Prosecutors. [Utah Code § 77-2-5(1), (4) (2012),Utah Code § 77-2a-3(6)
        (2012)]
      - Magistrates. [Utah Code § 77-2a-3(1) (2012)]
      - Defendants. [Utah Code § 77-2-5(1), (4) (2012),Utah Code § 77-2a-3(6)
        (2012)]

    * Types of pretrial diversion authorized by state law:

      - Plea in abeyance. [Utah Code § 77-2a-3]
      - Counties are authorized to implement any pretrial diversion program,
        subject to statutory restrictions. [Utah Code § 77-2-5]

    * Types of defendants eligible for pretrial diversion (program eligibility
      may be satisfied by different combinations of criteria):

      - Defendants who voluntarily agree to participation. [Utah Code §
        77-2-9(1) (2012),Utah Code § 77-2a-3 (2012)]
      - Counties are authorized to determine the eligibility of defendants,
        subject to statutory restrictions. [Utah Code § 77-2-9(1) (2012),Utah
        Code § 77-2a-3 (2012)]

    * Offenses eligible for pretrial diversion:

      - All misdemeanors. [Utah Code § 77-2-9(1) (2012),Utah Code § 77-2a-3
        (2012)]
      - Low-level felonies. [Utah Code § 77-2-9(1) (2012),Utah Code § 77-2a-3
        (2012)]
      - Counties are authorized to determine which offenses are eligible for
        pretrial diversion, subject to statutory restrictions. [Utah Code §
        77-2-9(1) (2012),Utah Code § 77-2a-3 (2012)]

    * Offenses ineligible for pretrial diversion:

      - Capital felonies. [Utah Code § 77-2-9(1) (2012),Utah Code § 77-2a-3
        (2012)]
      - 1st degree felonies. [Utah Code § 77-2-9(1) (2012),Utah Code § 77-2a-3
        (2012)]
      - Domestic violence offenses. [Utah Code § 77-2-9(1) (2012),Utah Code §
        77-2a-3 (2012)]
      - Any motor vehicle offense involving alcohol or drugs. [Utah Code §
        77-2-9(1) (2012),Utah Code § 77-2a-3 (2012)]
      - Felonies involving the use of a motor vehicle. [Utah Code § 77-2-9(1)
        (2012),Utah Code § 77-2a-3 (2012)]
      - Sexual offenses against a child under 14 years old. [Utah Code §
        77-2-9(1) (2012),Utah Code § 77-2a-3 (2012)]
      - Negligent use of a commercial motor vehicle resulting in death. [Utah
        Code § 77-2-9(1) (2012),Utah Code § 77-2a-3 (2012)]
