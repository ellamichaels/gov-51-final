Legal Context - Washington

Non-Custodial Promise to Appear Instead of Custodial Arrest

  * Citation in Lieu of Arrest

    * Can citations be issued as an alternative to custodial arrest for criminal
      offenses?

      - Yes. [Wash. Crim. R. Ct. Ltd. J. 2.1]

    * Offenses eligible for citations instead of arrest:

      - Gross misdemeanors. [Wash. Crim. R. Ct. Ltd. J. 2.1(b)(1)]
      - All misdemeanors. [Wash. Crim. R. Ct. Ltd. J. 2.1(b)(1)]

    * Circumstances under which citations shall not be issued:

      - Arrestee poses a danger to self or others. [Wash. Crim. R. Ct. Ltd. J.
        2.1(b)(2)]
      - There is reasonable suspicion that the arrestee will fail to appear in
        court. [Wash. Crim. R. Ct. Ltd. J. 2.1(b)(2)]
      - Arrestee has previously failed to respond to a notice to appear. [Wash.
        Crim. R. Ct. Ltd. J. 2.1(b)(2)]
      - Arrestee cannot be identified or refuses to identify him or herself.
        [Wash. Crim. R. Ct. Ltd. J. 2.1(b)(2)]

    * Are citations presumptive for eligible offenses?

      - No. [Wash. Crim. R. Ct. Ltd. J. 2.1]

    * Authorities who may issue citations:

      - The arresting officer. [Wash. Crim. R. Ct. Ltd. J. 2.1(b)(1)]
      - Other authorized personnel. [Wash. Crim. R. Ct. Ltd. J. 2.1(b)(1)]

  * Sentencing Options

    * Legal status of marijuana:

      - Possession of an amount less than 1 oz. is legalized. [Wash. Rev. Code §
        69.50.4013(3) (2013)]

  * Clearing a Criminal Record

    * Circumstances under which an arrest record may be cleared (qualification
      requirements may be satisfied by different combinations of the criteria):

      - Please note, the law(s) governing expungement have changed since the
        time of this study period. Please refer to the current statutes for most
        up-to-date information.. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code
        § 10.97.060]
      - A petition or request has been filed and accepted. [Wash. Admin. Code §
        446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - The court has ordered or approved an expunction request. [Wash. Admin.
        Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - The petitioner is not involved in any pending criminal procedures.
        [Wash. Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - The petitioner was acquitted at trial on all charges where expunction is
        sought. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060
        (2013)]
      - At least 2 years have passed since the record became non-conviction data
        due to a disposition favorable to the defendant. [Wash. Admin. Code §
        446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - At least 3 years have passed since the arrest or issuance of a citation
        or warrant, and the relevant charges did not result in a conviction.
        [Wash. Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - The disposition was not a deferred prosecution or similar diversion, or
        if so, the criminal justice agency has authorized expunction despite
        this disposition. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code §
        10.97.060 (2013)]
      - The petitioner has no prior felony or gross misdemeanor convictions, or
        if so, the criminal justice agency has authorized expunction despite
        these convictions. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code §
        10.97.060 (2013)]
      - The petitioner has not been arrested or charged with another offense
        subsequent to the arrest or charge where expunction is sought, or if so,
        the criminal justice agency has authorized expunction despite this
        charge. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060
        (2013)]
      - The charges where expunction is sought were not prosecuted. [Wash.
        Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - The charges where expunction is sought were dismissed. [Wash. Admin.
        Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]

      - Termination of parental rights. [Wash. Rev. Code § 13.34.180 (2013)]
      - Deportation. [Wash. Rev. Code § 9.94A.685 (2013)]

  * Employment

    * Does state law restrict employment application questions regarding prior
      arrests or convictions?

      - Yes. [Wash. Admin. Code § 162-12-140(3)(b), (3)(d)]

    * Aside from health care and law enforcement, are there other licensed
      professions that can exclude persons with arrest or conviction records?

      - Yes. [Wash. Admin. Code § 388-113,Wash. Rev. Code § 74.08.090
        (2013),Wash. Rev. Code § 74.39A.056 (2013)]

    * Circumstances under which employers may use criminal records when making
      employment decisions:

      - Arrest record inquiries are limited to arrests occurring within the last
        10 years and only include whether or not charges are still pending, have

        up-to-date information.. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code

      - A petition or request has been filed and accepted. [Wash. Admin. Code §

        Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - The petitioner is not involved in any pending criminal procedures.
        [Wash. Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - The petitioner was acquitted at trial on all charges where expunction is
        sought. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060
        (2013)]
      - At least 2 years have passed since the record became non-conviction data
        due to a disposition favorable to the defendant. [Wash. Admin. Code §
        446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - At least 3 years have passed since the arrest or issuance of a citation
        or warrant, and the relevant charges did not result in a conviction.
        [Wash. Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]

        if so, the criminal justice agency has authorized expunction despite
        this disposition. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code §
        10.97.060 (2013)]
      - The petitioner has no prior felony or gross misdemeanor convictions, or
        if so, the criminal justice agency has authorized expunction despite
        these convictions. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code §
        10.97.060 (2013)]
      - The petitioner has not been arrested or charged with another offense
        subsequent to the arrest or charge where expunction is sought, or if so,
        the criminal justice agency has authorized expunction despite this
        charge. [Wash. Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060
        (2013)]
      - The charges where expunction is sought were not prosecuted. [Wash.
        Admin. Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
      - The charges where expunction is sought were dismissed. [Wash. Admin.
        Code § 446-16-025,Wash. Rev. Code § 10.97.060 (2013)]
